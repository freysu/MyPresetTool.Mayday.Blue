使用指南
  1. 1. 选择音频文件
    • audio_file 上传音频文件
      点击“选择文件”，上传一首音频（支持 MP3 、 WAV 等格式）。
      音频的节奏和旋律将决定颜色的变化。
    • description 添加歌词文件（可选）
      上传 .lrc 格式的歌词文件，帮助系统理解音乐情感，自动匹配颜色。
  2. 2. 配置颜色主题
    • add_circle 选择基础颜色
      挑选代表歌曲情感的基础色。
    • add_circle 设置动态颜色范围
      确保每个颜色范围的总和为100%。
    • save 配置完成后，请点击 保存 。
  3. 3. 生成代码并获取结果
    • play_arrow 生成代码
      点击 生成 ，获取预设的颜色代码。
    • content_copy 复制或下载
      点击 复制 预览效果，或点击 下载 保存文件。
  4. 4. 示例配置
    • 例如，低频部分使用100%白色，主歌使用30%黄色和70%橙色，高潮使用30%橙色和70%红色。
